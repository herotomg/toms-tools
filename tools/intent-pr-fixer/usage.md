# Intent PR Fixer

Install the Intent specialist that fixes unresolved PR review comments in one pass.

## Before you start

- Install `gh-unresolved` first. This tool depends on it.
- Open Intent on the PR branch you want to fix.

## In Intent

1. Create an agent.
2. Pick the `PR Fixer` specialist.
3. Ask it to fix unresolved CR comments on the current branch.
