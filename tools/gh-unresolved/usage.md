# gh unresolved

Install a `gh` alias for reviewing unresolved pull request comments without leaving the terminal.

## Commands

- `gh unresolved` shows unresolved comments for the PR attached to the current branch.
- `gh unresolved 123` shows unresolved comments for PR `#123`.

## Setup

- Sign in to GitHub CLI first.
- Follow the setup guide: https://www.notion.so/gh-unresolved-setup-guide-34f6662456348037aacef0d70d99b3ac
